💀 ESCAPE ROOM — Horror Web Experience
Escape Room is an immersive, state-driven horror puzzle game built entirely with modern front-end technologies. It challenges the boundaries of what can be achieved in a browser without heavy game engines, featuring algorithmic audio synthesis, real-time tab synchronization, and high-fidelity CSS animations.

🌑 Features
Algorithmic Soundscape: Uses the Web Audio API to synthesize haunting ambient drones and UI sound effects in real-time.

Atmospheric UI: A "Glassmorphic" dark-themed interface with custom CSS keyframes including blood-drip effects and portal transitions.

Serverless Multiplayer Sync: Implements a custom synchronization engine using the Web Storage API (localStorage events) to mirror game states across multiple browser tabs.

Interactive Puzzles: Custom-built drag-and-drop physics and password entropy validation for an engaging user onboarding experience.

Responsive & Lightweight: Zero external dependencies (No React, No jQuery). Just pure HTML, CSS, and Vanilla JavaScript.

🛠️ Tech Stack
Structure: HTML5 (Semantic Layouts)

Styling: CSS3 (Variables, Flexbox/Grid, Keyframe Animations)

Logic: Vanilla JavaScript (ES6+)

APIs: - Web Audio API (Sound synthesis)

Web Storage API (Multiplayer state sync)

DOM Manipulation API
